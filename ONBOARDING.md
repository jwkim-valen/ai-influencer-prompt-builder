# 시작 가이드

---

## 1. 처음 설치 (최초 1회)

터미널을 열고 아래 명령어를 입력하세요:

```bash
git clone https://github.com/jwkim-valen/ai-influencer-prompt-builder
```

이후 Claude Code 앱에서 `Open Folder`로 `ai-influencer-prompt-builder` 폴더를 선택하면 바로 사용 가능합니다.

---

## 2. 업데이트 받기

새로운 캐릭터나 기능이 추가됐다는 공지가 오면, 해당 폴더에서 아래 명령어를 입력하세요:

```bash
git pull
```

내 작업 내용은 그대로 유지되고, 업데이트된 내용만 반영됩니다.

---

## 3. 사용 규칙 (중요)

아래 공통 파일은 **수정하지 말아 주세요.**
업데이트 시 충돌이 생길 수 있습니다.

```
수정 금지
├── CLAUDE.md
├── master_prompt.md
├── setup-prompt.md
├── skills/
└── characters/
```

프롬프트 생성 및 저장은 자유롭게 하셔도 됩니다.
새로 생성된 파일은 `git pull` 시에도 영향을 받지 않습니다.

---

## 4. 자세한 사용법

`GUIDE.md` 파일을 참고하거나, Claude Code 채팅창에서 아래를 입력하세요:

```
GUIDE.md 열람
```
